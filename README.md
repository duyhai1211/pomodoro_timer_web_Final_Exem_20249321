# Pomodoro 타이머 웹앱

이 프로젝트는 HTML, CSS, JavaScript를 이용해 만든 간단한 Pomodoro 타이머입니다. 사용자는 집중 시간(25분)과 휴식 시간(짧은 5분, 긴 15분)을 선택하여 작업과 휴식을 효율적으로 관리할 수 있습니다.

## 주요 기능

* 25분 집중 타이머 기본 제공
* 일시정지, 초기화 기능
* 짧은 휴식(5분), 긴 휴식(15분) 모드 지원
* 현재 상태 표시: "집중 시간", "짧은 휴식", "긴 휴식"
* 알림창을 통한 종료 알림

## 요구 사항

* 최신 웹 브라우저 (Chrome, Firefox, Edge 등)
* 별도의 서버나 설치 없이 로컬에서 실행 가능
